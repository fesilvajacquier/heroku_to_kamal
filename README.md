# De Heroku a Kamal

Presentación para la _meetup_ de [**RubySur**](https://ruby.com.ar/) — fecha por confirmar.

## Descripción corta

Crónica de la migración de una app Rails multi-tenant desde Heroku a un VPS con Kamal 2, usando IA como pair programmer.

## Ver las slides

[fesilvajacquier.github.io/heroku_to_kamal](https://fesilvajacquier.github.io/heroku_to_kamal/)

## Correr localmente

```bash
yarn install
yarn start
```

Abrir [localhost:8000](http://localhost:8000). Presionar `S` para ver las speaker notes.
